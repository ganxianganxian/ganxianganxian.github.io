搭建个人网站

在熟悉语法之后，就可以根据自己需要构建个人网站了。

@*********************************************************@

部署网站

使用github部署网站。

首先注册github账号.下载并安装git.

然后打开git-bash，进行SSH KEYS配置。

在git-bash输入$ ssh-keygen -t rsa -C "邮件地址@youremail.com"，邮箱地址改为自己的邮箱。

一直回车，系统提示设置密码，此处可以不设密码，也可以设置个人密码。

继续回车,设置SSH KEYS成功。

此时会在默认路径生成一个.ssh文件夹。

打开文件夹里的id_rsa.pub文件，并将其内容复制。

回到github页面，打开右上角的account settings，选择左边的SSH and GPG keys，再选NEW SSH KEYS。

title随便填，将刚才复制的id_rsa.pub文件内容粘贴到key处，最后点击Add SSH key即可。

回到git-bash进行测试并设置用户信息

输入$ ssh -T git@github.com，跳出信息后再输入yes.

输入$ git config --global user.name "yourname"//用户名

$ git config --global user.email "yourmail@gmail.com"//填写自己的邮箱

则成功配置了SSH KEY。

@*********************************************************@

创建GITHUB PAGE

回到github页面，选择new repository来新建仓库。仓库命名为name.github.io的形式，我这里命名为ganxianganxian.github.io

仓库创建成功后，打开仓库，选择setting，往下拉，点击Launch automatic page generator按钮。

在新页面的下面点击continue to layouts按钮。

根据自己喜好选择网站模板，点击publish page即成功发布网站。

最后进行远程仓库克隆，同步到本地仓库（自己新建的文件夹），再将自己设计好的网站文件粘贴到本地仓库，在git-bash上进行添加并发布，即可随时自己更新网站。
